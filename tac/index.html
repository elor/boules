Speicherstand kann nicht geladen werden.

Das im Browser gespeicherte Turnier konnte nicht geladen werden. Die Ursache kann in einem Fehler in Tuvero, einem extrem alten Speicherstand oder der falschen Tuvero-Variante liegen. Der Speicherstand wurde nicht überschrieben. Es gibt jetzt folgende Möglichkeiten:
  • Vielleicht wird im Hintergrund schon eine neue Version von Tuvero heruntergeladen, mit der der Speicherstand geöffnet werden kann. Wenn eine neue Version heruntergeladen wurde, kann man sie durch Neuladen starten.
  • Offline-Versionen erhalten keine Updates. Auf https://www.tuvero.de ist immer die neueste Version verfügbar.
  • Wurde die richtige Tuvero-Variante geöffnet? Eine Übersicht ist auf https://www.tuvero.de verfügbar.
  • Wenn der Turnierstand nicht mehr benötigt wird, kann er gelöscht werden. Zum Löschen muss #reset geöffnet werden.
  • Für technische Details kann man die JavaScript-Konsole öffnen (Chrome: Strg+Shift+J). Dort stehen mögliche Ursachen.
  • Wenn der Fehler weiterhin besteht, dann bitte ich um Kontakt, z.B. über erik@tuvero.de. Bitte fügt auch die Ausgabe der JavaScript-Konsole bei.
Informationen zu neuen Versionen und behobenen Fehlern stehen auf http://blog.tuvero.de.

Konnte eine Skriptdatei nicht laden. Sollte der Fehler auch bei wiederholtem Neuladen der Seite bestehen bleiben, bitte ich um eine Nachricht an erik@tuvero.de.

Erste Schritte

So führst du ein Turnier durch:
  1. Trage links den Turniernamen ein und erstelle ein neues Turnier
  2. Trage die Teilnehmer im Teilnehmer-Tab ein
  3. Wähle und starte ein Turniersystem im Turniersysteme-Tab
  4. Trage die Spielergebnisse im Turnierverlauf-Tab ein
  5. Sieh die Platzierung im Platzierungen-Tab ein
  6. Drucke Begegnungen und Ergebnisse über den Drucker-Knopf
